Bismuth - Run Online

Bismuth is a familiar, expressive, safe language that tries to keep code predictable and easy to refactor. It also intends to reduce bugs by making program intent and behavior machine-checkable. Bismuth will initially target JavaScript, so that it may be easily run in the browser. Then, it will be extended to target C.

Implementation

The Bismuth compiler is implemented in TypeScript, which allows it to run on this page.
